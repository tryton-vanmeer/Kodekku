<div align="center">
<h1>Kodekku (コーデック Codec)</h1>
CLI tool for getting codec info about video files

![screenshot](./assets/demo.gif)

</div>

## Usage

```shell
kodekku --help
```

## Development

```shell
# requirements
# https://github.com/zmwangx/rust-ffmpeg/wiki/Notes-on-building
dnf install ffmpeg-devel

# build
cargo build

cargo run -- <ARGS>
```
